# Random Map Generator

A simple map generator to make a random map each time.
Currently it only makes a simple plans with rivers and small hills

## Plans:

- Add biomes
- Add Modifiers
- Add a simple ui to change properties
- Generate landmarks such as trees, rocks, flowers, etc
